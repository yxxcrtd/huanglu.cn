Docker 安装 Redis

一，Docker 安装 Redis

1，安装：

docker pull redis

2，启动：

docker run --name redis -p 6379:6379 --restart=always -d redis redis-server --appendonly yes --requirepass "123456"

3，启动第N个：

docker run --name redis6380 -p 6380:6379 --restart=always -d redis redis-server --appendonly yes --requirepass "123456"

4，进入容器：

docker exec -it "容器ID" /bin/bash
