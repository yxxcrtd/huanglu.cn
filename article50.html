IDEA 远程调试

一，在本地配置 IDEA

1，菜单栏 Run -> Edit Configurations

2，添加一个：Remote

3，修改 IP 地址、修改端口号

4，拷贝命令行参数：-agentlib:jdwp=transport=dt_socket,server=y,suspend=n,address=5005


二，在服务端启动：远程服务

java -jar -agentlib:jdwp=transport=dt_socket,server=y,suspend=n,address=5005 /Users/young/Workspaces/home/target/home-0.0.1-SNAPSHOT.jar


三，在本地启动 IDEA

1，控制台只输出一行：

Connected to the target VM, address: '192.168.3.203:5005', transport: 'socket'


四，本地浏览器访问

http://192.168.3.203:5005
