Visual Studio Code 下的 Java 开发

“ VSCode 是由微软开发，支持 Windows、Linux 和 MacOS 等操作系统。它是开放源代码的代码编辑器，同时它又极其快速和轻量级！

一，Mac 下安装方法

brew install visual-studio-code


二，常用设置

1，隐藏某些文件：Settings -> 输入：file exclude -> 当个文件是：.idea；文件夹是：**/out

2，禁用/启用小地图：Settings -> Text Editor -> Minimap -> Enabled

3，设置微调与建议：Settings -> Text Editor -> Suggestions

4，遥测技术影响性能：Settings -> Preferences -> Text Editor -> Telemetry

5，禁用/启用代码透镜：Settings -> Preferences -> Text Editor -> Code lens


三，安装 VS Code 插件

1，Java Extension Pack

2，Spring Boot Extension Pack

3，Lombok

4，Gradle Extension Pack

5，Live Server Preview


四，配置 JDK

问题：VSCode 升级后会弹出：很抱歉，激活面向 Java 的 IntelliCode 支持时遇到问题。有关详细信息，请查看“针对 Java 的语言支持”和 “VS IntelliCode” 输出窗口

4，将插件：Language Support for Java by Red Hat 退到 0.64.1 版本


五，


六，打开 vscode 创建一个普通的 Java Project

1，安装插件：Java Extension Pack

2，设置 JDK：Code -> Preferences -> Settings -> User -> Extensions -> Java -> 找到 Edit in settings.json，添加 Java 的安装地址：

{

    "java.home": "/Library/Java/JavaVirtualMachines/openjdk-11.jdk/Contents/Home"

}

3，创建项目：command + shift + p，输入 creat -> Create Java Project -> 选择 -> No build tools -> 选择存储路径 -> 输入项目名称 -> 重新打开一个窗口 -> 打开项目中 src 下的 App.java

4，运行 Hello World！
