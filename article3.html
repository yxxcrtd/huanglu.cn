安装Mac下的包安装工具 Homebrew

1，终端执行：/usr/bin/ruby -e “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)”（该命令在官网获取：https://brew.sh）

2，安装 brew 增强，步骤1：brew tap phinze/cask

3，安装 brew 增强，步骤2：brew install brew-cask

4，查看安装版本：brew -v，出现：Homebrew 1.1.10 ...，表示安装成功！

5，软件更新：brew update && brew upgrade && brew cask upgrade && brew cleanup（建议加到终端的启动项中，这样能保证软件最新和清除安装后的文件包，保持系统最新。）

6，查找软件：brew search 或 info ‘需要安装的软件*’

7，更新软件：brew upgrade 或 reinstall redis

8，卸载软件：brew uninstall / remove ‘…’

9，查询已安装的软件：brew 或 brew cask list

10，清理安装包：brew cleanup

11，查看更新：brew outdated && brew cask outdated

12，不建议修改 Homebrew 的国内源，一样的慢！
