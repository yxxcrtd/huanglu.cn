Docker 安装 Oracle 12c

一，Docker 下载镜像

docker pull truevoly/oracle-12c


二，启动后初始化数据库文件：

docker logs -f ‘容器ID’


三，进入容器

docker exec -it ‘容器ID’ /bin/bash

进入数据库：sqlplus system/oracle@//localhost:1521/orcl

用户名：system

密码：oracle

测试： select * from dual;
