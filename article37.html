一，基本使用

1，切换用户：su - db2inst1

2，显示所有连接：db2 list application

3，强制关闭实例上的应用程序：db2 force application all

4，备份：db2 backup db pzpt to /home/tag

5，备份成功后显示：Backup successful. The timestamp for this backup image is : 20200401160847

6，在文件中中ll显示：PZPT.0.db2inst1.DBPART000.20200401160847.001

7，还原：db2 restore db pzpt from /home/tag taken at 20200401160847 into pzpt321（pzpt321是自定义的数据库名称）

8，还原成功后显示：DB20000I The RESTORE DATABASE command completed successfully.
