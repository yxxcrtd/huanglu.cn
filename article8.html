Git 基础

一，Linux CentOS 下安装 Git

1，安装依赖：yum -y install gcc perl-ExtUtils-MakeMaker curl-devel expat-devel gettext-devel openssl-devel zlib-devel

2，下载：wget https://www.kernel.org/pub/software/scm/git/git-2.10.1.tar.gz

3，解压：tar zxvf git-2.10.1.tar.gz

4，进入：cd git-2.10.1

5，编译：make prefix=/home/Git all

6，安装：make prefix=/home/Git install

7，添加环境变量：echo "export PATH=$PATH:/home/Git/bin" >> /etc/bashrc（在此文件上加一个：:/root/bin:/home/Git/bin）

8，使环境变量生效：source /etc/bashrc

9，验证：git --version，出现：git version 2.10.1，表示安装成功！(/home/Git/bin/git)


二，Mac 下安装 Git

1，brew install git

2，验证：git --version，出现：git version 2.10.1，表示安装成功！


三，删除文件夹下的所有 .git 文件

find . -name ".git"| xargs rm -Rf

终端执行：xcode-select --install
