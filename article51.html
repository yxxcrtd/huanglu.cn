IDEA 设置同步到 Github

一，在 GitHub 配置

1，在 GitHub 上新建一个空的仓库：mac-idea-settings

2，复制整个仓库的地址备用：https://github.com/yxxcrtd/mac-idea-settings


二，在 GitHub 生成 access token

1，登录 GitHub

2，在右上角点个人头像选择 Settings

3，在左边找到：Developer settings

4，Personal access tokens

5，Generate new token

6，需要输入 Github 密码

7，填写一个 token 的目的

8，接着勾选：repo、admin:repo_hook、delete_repo

9，点击：Generate token

10，最后拷贝生成的 token
