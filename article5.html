IDEA 插件 - JRebel 和 XRebel

安装 JRebel

1，在 IDEA 中安装插件：JRebel for IntelliJ（2020.1.0版本中已经添加了：XRebel）

2，激活：Team URL：https://jrebel.qekang.com/06b5c57d-457b-4fb8-a69e-3fc6d83ce38d

3，输入邮件地址，需要验证

4，启动：Run -> Active Rebel agent is JRebel. Specify ... > 选择需要使用的插件


安装 XRebel

1，下载：https://www.jrebel.com/products/xrebel/download

2，在 VM Options 添加：-javaagent:/home/xrebel.jar（注意解压后的 jar 文件目录）
