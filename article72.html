Docker 安装 Jenkins

一，Docker 下载镜像

docker pull jenkins

二，创建Jenkins的data文件：

mkdir -pv /home/docker/jenkins/data -m 777

三，启动：

docker run --name jenkins1 -p 58081:8080 -p 50001:50000 -v /home/docker/jenkins/data1:/var/jenkins_home jenkins

四，浏览器访问：

http://192.168.1.2:58081

五，查看密码

cat /var/jenkins_home/secrets/initialAdminPassword
