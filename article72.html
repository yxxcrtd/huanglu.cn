Visual Studio Code 下的 Java 开发

一，Mac 下安装方法

brew install visual-studio-code


二，安装 Open JDK 11 +

1，Mac 安装最新版本的 Java：

brew install java

2，配置环境变量：参考文章：JDK


三，安装 Maven

参考文章：Maven 基础


四，安装 Gradle

参考文章：Gradle 基础


五，打开 vscode 创建一个普通的 Java Project

1，安装插件：Java Extension Pack

2，设置 JDK：Code -> Preferences -> Settings -> User -> Extensions -> Java -> 找到 Edit in settings.json，添加 Java 的安装地址：

{

    "java.home": "/Library/Java/JavaVirtualMachines/openjdk-11.jdk/Contents/Home"

}

3，创建项目：command + shift + p，输入 creat -> Create Java Project -> 选择 -> No build tools -> 选择存储路径 -> 输入项目名称 -> 重新打开一个窗口 -> 打开项目中 src 下的 App.java

4，运行 Hello World！
