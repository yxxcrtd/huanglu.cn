Maven 使用 thirdparty 库

一，安装基于 2.x 版本的 Nexus（最新 3.x 版本的 TODO）

1，安装文档参考《Nexus 基础》


二，配置本地安装的 Maven

1，修改仓库路径：

<localRepository>/Users/young/repo</localRepository>

2，在 servers 节点中添加登录 Nexus 的账户

<server>

        <id>thirdparty</id>

        <username>admin</username>

        <password>admin123</password>

</server>

3，在 mirrors 节点中添加 Alibaba 的 公共私服

<mirror>

        <id>alimaven</id>

        <name>admin</name>

        <url>http://maven.aliyun.com/nexus/content/groups/public/</url>

        <mirrorOf>central</mirrorOf>

</mirror>

4，在 profiles 节点中配置：全局私服的配置信息

<profile>

        <id>central-profile</id>

        <repositories>

                <repository>

                        <id>maven-public</id>

                        <url>http://1.2.3.4:8081/repository/maven-public</url>

                        <releases>

                                <enabled>true</enabled>

                                <updatePolicy>always</updatePolicy>

                        </releases>

                        <snapshots>

                                <enabled>true</enabled>

                                <updatePolicy>always</updatePolicy>

                        </snapshots>

                </repository>

        </repositories>

</profile>

5，在 activeProfiles 节点中激活配置

<activeProfile>central-profile</activeProfile>


三，定义一个外部的 settings.xml （文件名随意）

1，定义一个外部的 settings.xml

<?xml version="1.0" encoding="UTF-8"?>

<settings>

        <localRepository>/Users/young/repo（要与第二步中的第1点相同）</localRepository>

        <mirrors>

                <mirror>

                        <id>nexus</id>

                        <url>central http://1.2.3.4:8081/nexus/content/repositories/thirdparty/</url>

                        <mirrorOf>central</mirrorOf>

                </mirror>

        </mirrors>

</settings>

2，在 IDEA 的 Maven 设置 User settings file 中 指定 settings.xml 位置


四，在 IDEA 中 deploy 到 thirdparty 库

1，在终端中执行：mvn clean deploy -DskipTests

2，Enjoy！
