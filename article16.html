解决 GitHub 授权的问题

1，检查 SSH keys，如果之前有的话，最好都删除，再将本地的 id_rsa.pub 重新上传上去

2，测试：ssh -T git@github.com，出现：Hi 用户名! You've successfully authenticated, but GitHub does not provide shell access. 表示成功了！

3，现在使用 SourceTree 测试

4，或者使用命令：git clone git@github.com: 用户名/******.git
