Docker 可视化管理工具 - Portainer & DockerUI

一，Portainer 的安装

1，下载镜像：docker pull docker.io/portainer/portainer

2，启动：docker run -d -p 9000:9000 --restart=always -v /var/run/docker.sock:/var/run/docker.sock --name prtainer-test portainer/portainer

3，访问：http://192.168.1.2:9000


二，Portainer 汉化

1，新建文件夹：/home/docker/public

2，将下载的 Portainer-CN 解压，放到 public 下（css、fonts、images、js 和 index.html）

3，启动：docker run -d -p 9001:9000 --restart=always --name prtainer -v /var/run/docker.sock:/var/run/docker.sock -v /home/docker:/data -v /home/docker/public:/public portainer/portainer

4，访问：http://192.168.1.2:9001


三，DockerUI 的安装

1，下载镜像：docker pull uifd/ui-for-docker

2，启动：docker run -it -d --name docker-web -p 9002:9000 -v /var/run/docker.sock:/var/run/docker.sock docker.io/uifd/ui-for-docker

3，访问：http://192.168.1.2:9002
