Docker 下安装 Nacos

1，Nacos 官网：https://nacos.io/zh-cn/

2，下载：docker pull nacos/nacos-server

3，检查：docker inspect fbf2aa0a0435

4，进入：docker exec -it ‘容器ID’ bash

5，创建容器：docker run --env MODE=standalone --name nacos -d -p 15434:8848 nacos/nacos-server

6，启动容器：docker start '容器ID'

7，访问：http://1.2.3.4:15434/nacos/
