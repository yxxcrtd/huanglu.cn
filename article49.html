Docker & Kubernetes（k8s）基础

一，安装 Docker Desktop

1，Mac 下用 brew 安装，也可以下载安装：https://www.docker.com/products/docker-desktop

brew cask install docker

2，安装后打开：Docker，就能看到 Docker 图标（最好登录 Docker Hub（yxxcrtd/C1），里面有教程等）

3，终端测试：docker run -dp 80:80 docker/getting-started

4，浏览器测试：http://localhost，自动跳转到：http://localhost/tutorial


二，镜像加速

1，访问阿里云官网：https://www.aliyun.com

2，登录（没有账户要注册）后搜索：容器镜像服务 （地址：https://cr.console.aliyun.com/cn-hangzhou/instances/mirrors），一般第一条搜索结果中点：立即开通，然后在左侧找：镜像加速器，里面有各种操作系统的操作文档。

3，拷贝自己的加速器地址：https://xxxxxx.mirror.aliyuncs.com

4，回到系统，在任务栏点击 Docker Desktop 应用图标 -> Perferences，在左侧导航菜单选择 Docker Engine，将刚才复制的加速器地址配置到 json 文件："registry-mirrors": ["https://xxxxxx.mirror.aliyuncs.com"]，如下图：

5，重启 Docker Desktop
