SourceTree 基础

一，安装

1，下载 SourceTree：https://www.sourcetreeapp.com

2，安装 SourceTree：需要有 Atlassia 的账户

3，在本地生成公钥和私钥：打开 SourceTree 右上角的终端：ssh-keygen，全部回车

ssh-keygen

4，将 cat ~/.ssh/id_rsa.pub 中的内容拷贝到 GitLab 的 Profile Settings -> SSH Keys 中，点一下 Title，保存

cat ~/.ssh/id_rsa.pub

5，授权（1，输入：yes，出现：Welcome to GitLab, @young!，说明你已经有访问 GitLab 的权限；2，默认的端口参数-p 22可以不加）

ssh -T -p 22 git@192.168.1.8

6，如果 GitLab 服务器上的 22 端口不能使用，则需要在本地的 .ssh 文件中添加一个 config 文件，内容如下：（上面的授权需要添加端口参数）

Host 192.168.1.8

Hostname 192.168.1.8

Port 10222

7，在 SourceTree 中配置连接 GitLab，下载代码


二，Mac 下安装 SourceTree

brew cask install sourcetree


三，添加本地已有项目到 GitHub

1，首先在 GitHub 上创建一个项目：https://github.com/yxxcrtd/demo（demo就是你创建的项目名称）

2，打开 SourceTree -> 项目列表 -> 新建 -> 添加已经存在的本地仓库 -> 选择已有的仓库文件夹 -> 打开

3，双击 demo -> 右上角 -> 设置 -> 远程仓库 -> 添加 -> 远程（仓库）名称：origin -> URL/路径：git@github.com:yxxcrtd/demo.git -> 确定

5，提交代码
