Spring Boot 之 热部署 + 自刷新 （DevTools + LiveReload）

一，IDEA端的设置：

1，添加依赖：(pom.xml)

2，添加配置：(application.yml)

3，IDEA 打开自动编译：Build, Execution, Deployment -> Compiler -> Make project automatically

4，注册配置：Command + option + shift + / -> registry -> compiler.automake.allow.when.app.running

5，IDEA 启动的时候会出现：LiveReload server is running on port 35729


二，浏览器端的设置：（目前 LiveReload 仅支持：Chrome、Firefox、Safari（2019年开始 Safari 已经不支持安装第三方插件了！））：

1，下载插件：http://livereload.com/extensions/

2，插件检测：http://localhost:35729/livereload.js?snipver=1（打开js，说明OK。）


三，Enjoy：

1，修改 java 文件的时候，spring-boot 会重启（这点没有 JRebel 优秀，JRebel只是重新加载修改后的 class）

2，修改 js、css、images图片、freemarker文件都可以在浏览器自动刷新
