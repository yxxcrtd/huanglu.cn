Docker 安装 Nginx

一，Docker 下载镜像

docker pull nginx


二，在宿主机上创建 Nginx 配置需要的4个文件夹和 nginx.conf 文件

mkdir -pv /home/docker/nginx/{conf,conf.d,html,log} -m 777


三，使用 Docker 启动 Nginx

docker run --name nginx -dp 80:80 -v /home/docker/nginx/conf/nginx.conf:/etc/nginx/nginx.conf -v /home/docker/nginx/log:/var/log/nginx -v /home/docker/nginx/html:/etc/nginx/html/ nginx

注意1：--name nginx 使用的镜像名称

注意2：最后的 nginx 是容器名称

注意3：静态文件夹映射的地址不是：/usr/share/nginx/html/
