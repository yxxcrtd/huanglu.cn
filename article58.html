Java 监控工具 JVisualVM 或 JConsole

一，配置 Tomcat 内存

1，在 catalina.sh 添加：JAVA_OPTS='-Xms5g -Xmx5g'


二，配置 JMX 连接

1，在 JAVA_OPTS="$JAVA_OPTS $JSSE_OPTS" 后添加：

JAVA_OPTS="$JAVA_OPTS -Dcom.sun.management.jmxremote -Dcom.sun.management.jmxremote.port=6666 -Djava.rmi.server.hostname=192.168.1.2 -Dcom.sun.management.jmxremote.ssl=false -Dcom.sun.management.jmxremote.authenticate=false"


三，本地终端启动：jvisualvm

1，远程 -> 右键：添加远程主机 -> 填写主机名即可：192.168.1.2

2，192.168.1.2 -> 右键：添加 JMX 连接 -> 在 连接 中补充端口：192.168.1.2:6666


四，本地终端启动：jconsole

1，远程进程 -> 填写：192.168.1.2:6666
